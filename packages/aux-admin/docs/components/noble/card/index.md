## Card 参数
| 参数名 | 类型 | 默认值 | 说明 | 跳转 Demo |

|  v-model    | Boolean  |   false   |   是否显示   |[基本用法](#基本用法)|           |

## Card 事件
| 事件名 | 说明 | 参数 |
| :---- | :---- | :--------- |
| close  |关闭回掉 |无|
| open |打开回掉 |无|
